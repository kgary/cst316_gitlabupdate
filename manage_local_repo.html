Managing Local Git Repository

Edit a file and check the status (git status)

  1. Open ButtonFrame.java in any text editor (donâ€™t go Eclipse yet) and add a few lines and delete a few lines (comment lines are fine).
  2. Check the status
    $git status

Add/Discard or Remove a file from/to git repository (git add, git checkout, git rm)

  1. Add ButtonFrame.java to the repository
    $git add ButtonFrame.java
  2. Remove ButtonFrame.java from the repository
    $git rm ButtonFrame.java
  3. Move ButtonFrame.java from the repository
    $git mv <from> <to>

Un-stage a file (git reset)

  1. Undo the changes
    $ git reset HEAD ButtonFrame.java

Check history of repo contents (git show)

  1. Display the hostory
    $ git show ButtonFrame.java

Find difference between repo contents (git diff)

  1. Display the difference
    $ git diff ButtonFrame.java
