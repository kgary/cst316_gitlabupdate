Using GUI-based tools for Git

I am a text-based person (and old) so I prefer to use the command-line, but you may like a GUI's features. It is an individual preference, but whether you use a GUI or not I believe it critically important to know the underlying foundations of git and to know what commands the GUI tool is issuing under the hood.

Using the egit eclipse plugin
  1. Add a plug in to eclipse found at http://www.eclipse.org/egit

Eclipse has a plugin named Egit which is supposed to be in the standard bundle, but if you do not have it you may choose to add it. Other popular IDEs have similar plugins.
