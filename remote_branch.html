Remote Branching

Create a local branch and push it to the shared repository

  1. Execute the below command to do so
    $ cd top/Jekyll/work1
    $ git checkout –b sidestuff
    $ <edit ButtonFrame.java>
    $ git status
    $ git commit –a
    $ git status
    $ git push origin sidestuff     # This pushes the new branch to the remote repository (origin)

Get the new branch to local repo

  1. Let’s pretend top/work1 on your local machine is a remote “shared” repo, and top/jekyll/work1 is where you, as a developer, are doing your coding. Do the following:
    $ cd top/jekyll/work1
    $ git remote –v
  2. Execute the following commands to create one more directory top/hyde, and clone work1 again:
    $ cd top
    $ mkdir hyde
    $ cd hyde
    $ git clone <absolute path to top/work1*gt
    $ cd work1
    $git remote –v
