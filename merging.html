Merging and Re-basing

Merge contents of a branch (git merge)

  1. Create branch named p2 and checkout.
    $ git checkout –b p2
  2. Edit ButtonFrame.java
  3. Add file to git
    $git add ButtonFrame.java
  4. Save the changes
    $ git commit –m “extra commit 1” ButtonFrame.java
  5. Store the changes on branch p1
    git commit –m “extra commit 1” ButtonFrame.java
  6. Check out master
    $ git checkout master
  7. Merge p2 to master
    $ git merge p2
  8. Check out master again
    $ git checkout master
  9. Merge p1 to master
    $ git merge p1

Apply all the changes from a branch (git rebase)

  1. Checkout branch p1.
    $ git checkout p1
  2. Edit ButtonFrame.java
  3. Add file to git
    $git add ButtonFrame.java
  4. Save the changes
    $ git commit –m “extra commit 1” ButtonFrame.java
  5. Edit ButtonFrame.java
  6. Save the changes
    $ git commit –m “extra commit 2” ButtonFrame.java
  7. Rebase master
    $ git rebase master
  8. Check out master
    $ git checkout master
  9. Merge p1 to master
    $ git merge p1
