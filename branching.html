Branching in Git

Create a branch (git branch)

  1. Create branch named p1.
    $ git branch p1
  2. Check the status
    $git status
  3. Checkout the branch p1
    $ git checkout p1
  4. Check the status
    $git status
  5. Display the branch p1
    $ git show p1
  6. Display the master
    $ git show master

Switch to branch or master

  1. Get the branch/master copy
    $git checkout <branch name>/ git checkout master

Check history of a branch

  1. Display branch
    $git show <branch name>

Check history of master copy of the repo (git show)

  1. Display the hostory
    $ git show master

Find difference between repo contents (git diff)

  1. Display the difference
    $ git diff
